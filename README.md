## REACT ON RAILS CALCULATOR

### Steps

```
- git clone https://github.com/fmauricios/react_on_rails_calculator.git
- cd react_on_rails_calculator
- bundle && yarn install
- rails db:drop db:create db:migrate db:seed
- yarn rails-server
```

### Tools

```
- Rails 5.0.2
- Ruby 2.4.1
- react_on_rails
- ES6
```
